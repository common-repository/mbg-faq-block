== Description ==
 
Add quick Q&A to your blog posts addressing popular questions. This plugin mark up your FAQ with FAQPage schema which is officially supported by Google.

FAQPage schema creates a prettier search snippet that attracts more attention and clicks.

You can use People Also Ask boxes to come up with popular questions to address in your content.
 
== Installation ==
 
1. Unpack plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
